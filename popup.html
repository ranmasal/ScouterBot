Troop Inventory via GitHub

Connect ScouterBot to a GitHub repository to track your troop's inventory — patches, awards, equipment, and more.

The bot can then tell you exactly what to order and when.

Would you like to connect your troop's inventory to GitHub?

Step 1 of 4: Do you have a GitHub account?

No problem! Here's how to create a GitHub account:

  1. Go to github.com/join
  2. Enter your email and create a password
  3. Verify your email address
  4. Come back here and click "I have an account"

Step 2 of 4: Create a repository

  1. Go to github.com/new
  2. Name it something like Troop123-Inventory
  3. Make it Private
  4. Check "Add a README file"
  5. Click Create repository

Step 3 of 4: Create a Personal Access Token

ScouterBot needs a token to read and write your inventory file.

  1. Go to github.com/settings/tokens/new
  2. Give it a name like ScouterBot-Inventory
  3. Select repo (Full control of private repositories)
  4. Click Generate token
  5. Copy the token immediately — you won't see it again!

Step 4 of 4: Test connection

Click "Test & Save" to verify everything works.

👍 No problem! You can connect your troop inventory anytime by going to Settings ⚙️ → Troop Inventory.

Until then, ScouterBot will still work great for answering questions about Scouting America policies and procedures.
